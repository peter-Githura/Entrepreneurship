Link to Google Sheets

Every sale you save here is pushed to a Google Sheet automatically, so you always have a live backup outside this tool.

One-time setup (5 min):
1. Create a blank Google Sheet, name it "Elfancy Vapes Sales".
2. Go to Extensions → Apps Script, delete any sample code.
3. Paste the script from the file google-apps-script-setup.gs I've shared alongside this tool.
4. Click Deploy → New deployment → type: Web app.
5. Set Execute as: Me and Who has access: Anyone, then Deploy.
6. Copy the Web app URL it gives you and paste it above.
7. If your Google Workspace account blocks "Anyone" access, repeat this with a personal Gmail account instead.

Use Pull if you're on a new device, cleared browser data, or want to bring in sales someone else logged straight into the sheet — it only adds rows you don't already have, never duplicates or overwrites.

Danger zone
